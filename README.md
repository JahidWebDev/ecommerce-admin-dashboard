# 🛒 MERN E-commerce Admin Dashboard (React + Tailwind CSS)

A modern and scalable **E-commerce Admin Dashboard** built with **React.js, Tailwind CSS, and MERN Stack (MongoDB, Express.js, Node.js)**.
This project helps developers and businesses manage **online stores, products, users, and orders** efficiently.

## 🚀 Features

* 📦 Product Management System (CRUD)
* 👤 User Management & Authentication
* 🧾 Order & Sales Management
* 📊 Admin Dashboard Analytics
* 🔐 JWT Authentication (Secure Login)
* ⚡ Fast UI with React & Tailwind CSS
* 🌐 REST API with Node.js & Express

---

## 🛠️ Tech Stack

### Frontend

* React.js (Admin Dashboard UI)
* Tailwind CSS (Modern Styling)
* Axios (API Requests)

### Backend

* Node.js (Server)
* Express.js (API)
* MongoDB + Mongoose (Database)

## ⚙️ Installation Guide

### 1. Clone Repository

```id="st2"
git clone https://github.com/JahidWebDev/ecommerce-admin-dashboard.git
```

### 2. Install Dependencies

#### Frontend

```id="st3"
cd client
npm install
```

#### Backend

```id="st4"
cd server
npm install
```

---

## ▶️ Run the Application

### Start Backend Server

```id="st5"
cd server
npm run dev
```

### Start Frontend

```id="st6"
cd client
npm start
```

---

## 🌐 Environment Variables

Create `.env` file inside **server/** folder:

```id="st7"
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
